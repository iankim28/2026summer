# lib/notebooks

Organised into topic folders; each contains its own `results/` directory.
Shared cross-experiment artifacts live in `image_samples/`. "Last used" is the
most recent file-modification date inside the folder (not just the last git commit).

| Folder | Last used | Research question | Main outputs |
|--------|-----------|-------------------|--------------|
| [`en_zh_multi_uni_attack/`](en_zh_multi_uni_attack/) | 2026-07-13 | Does attacking with **both** EN+ZH text (multilingual) beat a repeated EN-only attack (unilingual), and which defense is worth its inference cost? See [`PROTOCOL.md`](en_zh_multi_uni_attack/PROTOCOL.md) for the full spec. `multilingual/` and `unilingual/` each run `attack_comparison.ipynb` → `cam_defense.ipynb` (2-mod/4-mod GradCAM intersection) → `grid_defense.ipynb` (blind 1/2-patch occlusion); `cost_vs_performance.ipynb` aggregates all 9 runs. Sub-experiments: `_test_exhaustive_grid/` (confirms greedy 2-patch ≈ exhaustive 120-pair search, 11.0% vs 11.5% mean acc) and `_test_attention_defense/` (ViT self-attention as a **cheaper, better** GradCAM replacement — Attn-last hits 72.6% mean acc at cost 4 vs GradCAM's 33.1% at cost 6, with far less clean-image degradation). | `cost_vs_performance.png`; per-setup `results/attack/`, `results/cam_2mod/`, `results/cam_4mod/` (multilingual only), `results/grid_1patch/`, `results/grid_2patch/`; `_test_attention_defense/results/final_comparison.png` |
